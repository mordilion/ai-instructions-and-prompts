# Custom Functions (Company-Specific)

This folder is for **company-specific function patterns** that extend the core patterns in `.ai-iap/functions/`.

## Rules

- **ALWAYS** start new files from `.ai-iap/functions/_TEMPLATE.md`
- Put all metadata in YAML frontmatter
- After the YAML header: **code examples only**
- **NEVER** include installation commands in function files

## Suggested Files

- `INDEX.md` (optional): your company functions index
- `*.md`: real custom functions (kept local by default)
- `*.example.md`: shareable examples (tracked in git)
