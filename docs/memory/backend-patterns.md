# Backend patterns (this repository)

**No runtime API or server** ships from this repo.

## What “backend” means here

- **Rules and processes** for backend stacks live under `plugin/lib/rules/{java,python,php,...}` and `plugin/lib/processes/`
- **Authoring** follows general + language architecture/security/code-style files

## Related paths

- NestJS / Spring / Django / Laravel etc.: `plugin/lib/rules/<lang>/frameworks/`
- API documentation standards: `plugin/lib/rules/general/documentation/api.md`
- OpenAPI process entries: search `api-documentation-openapi` in `plugin/lib/config.json`
